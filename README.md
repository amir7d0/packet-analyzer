# packet-analyzer
A Python-based packet analyzer


### Features

- Capturing packets and save them in pcap file and print them onto cli
- Supports the following protocols :
	- Ethernet
	- ARP
	- ICMP
	- IPv4
	- IPv6
	- TCP
	- UDP
	- HTTP
	- DNS

## Requirement
- Python 3.x
	- socket
	- struct
	- textwrap
	- time
	- binascii
- Linux Operatin System
- Administrator Privileges "super user do"


## Run


```sh
sudo python3 app.py
```
